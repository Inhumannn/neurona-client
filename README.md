## Installation

### Prérequis

- npm >= v10.9.2
- Nodes.js >= v22.14.0

### Lancer le projet localement

```bash
git clone https://github.com/Inhumannn/neurona-client
cd nom-du-répertoire
npm install
npm run dev
```
